# Симуляция банкомата

## Постановка задачи

Реализовать программу-симулятор банкомата, которая позволяет пользователям:

1. **Авторизоваться** по номеру карты и PIN-коду
2. **Проверять баланс** счета
3. **Вносить деньги** на счет
4. **Снимать деньги** при достаточном балансе
5. **Переводить деньги** между пользователями

## Используемый паттерн проектирования

### Singleton

**Обоснование выбора:** в системе должен существовать только один экземпляр банкомата, управляющий всеми операциями. Множественные экземпляры банкомата могли бы привести к конфликтам данных (например, при одновременном изменении баланса одного пользователя из разных экземпляров).

**Реализация:**
- Приватный конструктор `ATMSingleton()`
- Статическое поле `_instance` для хранения единственного экземпляра
- Потокобезопасная инициализация через `lock`
- Публичное статическое свойство `Instance` для доступа
